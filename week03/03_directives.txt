Give MIPS directives to represent the following variables:

Note: when data is uninitialised, use .space directive

a) int u;
u:      .space 4


b) int v = 42;
v:      .word 42


c) char w;
w:      .space 1


d) char x = 'a';
x:      .byte 'a'


e) double y;
y:      .space 8


f) int z[20]; 
z:      .space 80
